# Lenovo One Pager

## Files
- `index.html` — main webpage
- `assets/styles.css` — styling
- `assets/script.js` — interactivity
- `assets/data.js` — centralized links and document mappings
- `docs_preview/` — HTML previews that open inside the modal
- `docs_original/` — original uploaded documents

## How to host
Upload the full folder contents to any static hosting service or web server.
Examples:
- GitHub Pages
- Netlify
- Vercel
- IIS / Apache / Nginx
- Shared hosting with static file support

## How to edit
- Update links in `assets/data.js`
- Update banner text directly in `index.html`
- Add more preview pages inside `docs_preview/`
- Keep original files in `docs_original/`

## Notes
- Preview pages were generated from the uploaded DOCX, XLSX, and TXT files for in-page viewing.
- Original files remain available through the "Open Original" buttons.
